# Docs

Some docs and utilities in developing this module

## 中文描述 ZH

1. `test_pd120_modulate.ipynb` 记录了一些 PD 120 的规范，包含前缀、VIS Code 和数据部分，以及一个生成过程实现；`pd120_encode.py` 是其简化的生成代码，效果相同；`pd120_challenge_gen.py` 是一个较差质量的信号生成，用于测试稳定性。
2. `read_img_check.ipynb` 读取 `main` 解码输出的 `.raw` 文件，查看数据情况
3. `read_wav_check.ipynb` 是一些读 `.wav` 或者测试代码，是临时的代码
4. `compare_.png` 是解码效果对照图
5. `Space_Comms...._sound.wav` 测试音频来自于 [ISS SSTV Audio Recordings](https://spacecomms.wordpress.com/iss-sstv-audio-recordings/)
6. `No-Signal-TV` 相关的内容从 PDF 生成
